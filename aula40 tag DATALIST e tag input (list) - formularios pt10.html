O atributo list refere-se a um elemento < datalist> que contém opções predefinidas para um elemento < input>.

A tag < datalist> especifica uma lista de opções predefinidas para um elemento < input>. A tag < datalist> é usada para fornecer um recurso "preenchimento automático" nos elementos < input>. Os usuários verão uma lista suspensa de opções predefinidas à medida que inserem dados. Use o atributo list do elemento < input> para vinculá-lo a um elemento < datalist> pelo id do datalist.
