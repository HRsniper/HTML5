A tag < meter> define uma medida escalar dentro de um intervalo conhecido ou de um valor fracionário. Isso também é conhecido como medidor. Exemplos: uso do disco, relevância de um resultado da consulta etc. Nota: A tag < meter> não deve ser usada para indicar progresso (como em uma barra de progresso). Para barras de progresso, use a tag < progress> . Dica: sempre adicione a tag < label> para obter as melhores práticas de acessibilidade!,label link com for no id do meter.
