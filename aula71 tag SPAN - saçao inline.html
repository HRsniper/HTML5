A < span> tag é um contêiner embutido usado para marcar uma parte de um texto ou parte de um documento. A < span> tag não fornece nenhuma alteração visual por si só, mas quando está marcada, você pode estilizá-la com CSS ou manipulá-la com JavaScript. Ele deve ser usado somente quando nenhum outro elemento semântico for apropriado. span é muito parecido com o elemento div, entretando div tem o display block e span tem o display inline.
