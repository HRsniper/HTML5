Aula 1 - Codificação básica

< !DOCTYPE html > Instrução para o navegador que diz qual versão de HTML estamos trabalhando

< html > < /html > Tag raiz, container para tudo conteudo da pagina

< head > < /head > (contido por < html >) cabeça da pagina, container para outras tag como link ,css,script e etc

< title > < /title > (contido por < head >) titulo da pagina //categoria metadado//

< body > < /body > (contido por < html >) corpo da pagina //categoria seção raiz//
