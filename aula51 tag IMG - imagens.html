HTML Tags HTML Tags Cheatsheet

A tag < img> define uma imagem em uma página HTML. A tag < img> possui dois atributos obrigatórios: src e alt. Nota: As imagens não são tecnicamente inseridas em uma página HTML, as imagens são vinculadas às páginas HTML. A tag < img> cria um espaço de espera para a imagem referenciada.
