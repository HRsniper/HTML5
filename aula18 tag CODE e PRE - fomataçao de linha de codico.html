A tag < code > serva para destacar, marcar um texto como sendo parte de um código de programação. A tag < pre > aplica a formatação na renderização assim como o texto foi digitado no arquivo HTML, com todos seus espações e enter.

#include < iostream > using namespace std; int main() { cout < < "Hello World" < < endl; return 0; }
