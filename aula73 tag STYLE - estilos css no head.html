A tag style> é usada para definir informações de estilo para um documento HTML. Dentro do elemento style>, você especifica como os elementos HTML devem renderizar em um navegador. Cada documento HTML pode conter várias tags style>. Dica: Para vincular a uma folha de estilos externa, use a tag < link> .
