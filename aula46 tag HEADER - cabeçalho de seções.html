tag de seção header, que representa a área de cabeçalho do context ou seção onde está inserida. O elemento < header> representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Um elemento < header> normalmente contém: um ou mais elementos de cabeçalho (< h1> - < h6>) logotipo ou ícone informação de autoria Você pode ter vários elementos < header> em um documento. Nota: Uma tag < header> não pode ser colocada em um elemento < footer>, < address> ou outro < header>.
