O elemento < select> é usado para criar uma lista suspensa. As marcas < option> dentro do elemento < select> definem as opções disponíveis na lista. Dica: O elemento < select> é um controle de formulário e pode ser usado em um formulário para coletar entrada do usuário.

A tag < option> define uma opção em uma lista de seleção. Os elementos < option> vão para dentro de um elemento < select> ou < datalist> . Nota: A tag < option> pode ser usada sem nenhum atributo, mas você geralmente precisa do atributo value , que indica o que é enviado ao servidor. Dica: Se você tiver uma longa lista de opções, poderá agrupar opções relacionadas com a tag < optgroup> .

A tag < optgroup> é usada para agrupar opções relacionadas em uma lista suspensa. Se você tiver uma longa lista de opções, é mais fácil gerenciar grupos de opções relacionadas para um usuário.
