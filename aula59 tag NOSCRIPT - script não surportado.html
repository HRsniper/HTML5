A tag define uma seção de html a ser inserida se um tipo de script não é suportado pela página ou se o script está desativado no navegador. O elemento < noscript> pode ser usado em < head> e < body>. Quando usado dentro do elemento < head>: < noscript> deve conter apenas elementos < link>, < style > e < meta>. Dica: Também é uma boa prática usar a tag de comentário para "ocultar" scripts de navegadores sem suporte para scripts do lado do cliente (para que eles não sejam exibidos como texto sem formatação). Fora dos head elementos, se o script estiver desativado para o noscript elemento O noscript modelo de conteúdo do elemento é transparente. Fora dos head elementos, se o script estiver ativado para o noscript elemento O noscript elemento deve conter apenas texto, exceto que o texto deve ser tal que a execução do algoritmo.
