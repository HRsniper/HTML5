HTML Tags HTML Tags Cheatsheet

A tag < form> é usada para criar um formulário HTML para entrada do usuário.
O elemento < form> pode conter um ou mais dos seguintes elementos de formulário:
< input>
< textarea>
< button>
< select>
< option>
< optgroup>
< fieldset>
< label>
< output>

A tag < input> especifica um campo de entrada onde o usuário pode inserir dados. Os elementos < input> são usados ​​em um elemento < form> para declarar controles de entrada que permitem que os usuários insiram dados. Um campo de entrada pode variar de várias maneiras, dependendo do atributo type.
Nota: O elemento < input> está vazio, contém apenas atributos.
Dica: Sempre use a tag < label> para definir rótulos para < input type = "text">, < input type = "checkbox">, < input type = "radio">, < nput type = "file"> e < input type = "password">.

A tag < label> define um rótulo para vários elementos: < input type = "color">

O atributo for da marca < label> deve ser igual ao atributo id do elemento relacionado para uni-los. Dica: Um rótulo é vinculado a um elemento usando o atributo "for" ou colocando o elemento dentro do elemento < label>.
