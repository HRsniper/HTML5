A tag < ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética, algarismos romanos. Use a tag < li> para definir itens da lista.

A tag < li> define um item da lista. A tag < li> é usada em listas ordenadas ( < ol> ), em listas não ordenadas ( < ul> ) e em listas de menus ( < menu> ).
