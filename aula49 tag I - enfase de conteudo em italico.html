aplicação em ênfase em conteúdos,A tag < i> define uma parte do texto em uma voz ou humor alternativo. O conteúdo da tag < i> geralmente é exibido em itálico. A tag < i> pode ser usada para indicar um termo técnico, uma frase de outro idioma, um pensamento ou um nome de navio, etc. Use o elemento < i> apenas quando não houver um elemento semântico mais apropriado, como:
< em> (texto enfatizado)
< strong> (texto importante)
< marca> (texto marcado / destacado)
< cite> (o título de uma obra)
< dfn> (um termo de definição)
