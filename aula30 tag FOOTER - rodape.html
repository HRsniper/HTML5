HTML Tags HTML Tags Cheatsheet

A tag < footer> define um rodapé para um documento ou seção. Um elemento < footer> deve conter informações sobre o elemento que o contém. Um elemento < footer> normalmente contém:
> informação de autoria
> Informações sobre direitos autorais
> Informações de Contato
> Mapa do site
> voltar ao topo links
> documentos relacionados
Você pode ter vários elementos < footer> em um documento.
Dica: As informações de contato dentro de um elemento < footer> devem estar dentro de uma tag < address> .
