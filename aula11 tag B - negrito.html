HTML Tags Cheatsheet

A tag < b > especifica texto em negrito sem nenhuma importância extra.
Nota: De acordo com a especificação do HTML 5, a tag < b > deve ser usada como ÚLTIMO resort quando nenhuma outra tag for mais apropriada. A especificação do HTML 5 indica que os títulos devem ser indicados com as tags < h1 > a < h6 > , o texto enfatizado deve ser indicado com a tag < em > ,
o texto importante deve ser indicado com a tag < strong > e o texto marcado / destacado deve usar a tag < mark >.
Dica: Você também pode usar a propriedade "font-weight" do CSS para definir texto em negrito.
